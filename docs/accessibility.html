Accessibility

Boost theme is built with accessibility in mind.

  • Color contrast
    • The contrast ratio between text and background is within the AA level (4.5:1).
  • Links
    • Links are underlined by default, except when theyâ€™re part of specific components.
  • Headings structure
    • All sections have a heading element (sometimes visually hidden).
    • Headings do not skip levels when descending.
  • Images
    • All images incorporate an alt attribute.
  • Forms
    • Forms have field labels associated with the form controls.
  • Keyboard navigation
    • The pages have a logical tab order.
    • Interactive controls and links are reachable using the keyboard.
    • The pages include a 'Skip to main content' link at the beginning.
  • Landmarks
    • HTML5 landmark elements are used to improve navigation (<main>, <nav>, etc.)
  • Valid W3C code
    • The code is fully compliant with the standards set by the World Wide Web Consortium.
